    • What is ol,ul and li in html?

      Answer:       OL means order list.It is a container tag that start with ol tagand ends with tag ol.It is used while making a list in orders.We can define the order if it is starts with 1 or A or I or a.

      UL means unorder list.It is a container tag that start with ul tag and ends with ul tag.It is used while making a list without any order.The item list will be mark by bullet pointer by default.We can change the type of bullet pointer like circle,square,none etc.

      LI means list items in html.It is used inside the ol and ul tag for making the items list with or without order.

      What is td and th in html?

      Answer:       TD means table data and TH means table heading in html.It is used while creating a table.We write the table heading inside th tag and table data inside td tag.
